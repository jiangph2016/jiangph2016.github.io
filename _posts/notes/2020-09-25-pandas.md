## DataFrame拼接

级联，参考<https://blog.csdn.net/Dorisi_H_n_q/article/details/82261971>


### 竖向拼接
两个dataframe拥有一个样的column,所以垂直拼接
```
df_train,df_test
df = pd.concat([df_train,df_test],ignore_index=True)
```

### 横向拼接

两个dataframe需要横向拼接

```
df_a,df_b
df = pd.concat([df_a,df_b],axis=0)
```
