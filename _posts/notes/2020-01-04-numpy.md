随用随记

```
import numpy as np
```

## 矩阵转置
```
a.T
```

## 矩阵乘法

```
a = np.mat([1,2,3])
b = np.mat([4,5,6])
ret = np.dot(a,b)
```

## 求矩阵的特征值和特征向量
a为特征值  
b为特征向量  
```
a,b = np.linalg.eig(ret)
```

## matrix 转换为数组

```
arr = ret.getA()
```

## 精度控制

对ret保留小数点后两位
```
np.around(ret,decimals = 2)
```

## 关闭科学技术法

```
np.set_printoptions(suppress=True)
```
