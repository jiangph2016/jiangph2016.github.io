## 正常使用

启动：`vncserver :1 -geometry=1280x720`

查看已有的vnc `vncserver -list`


## 安装

### Windows
序列号`77NVU-D9G5T-79ESS-V9Y6X-JMVGA`

### Ubuntu
```
apt-get install vnc4server
```

## 解释

### 运行模式

- Server Mode: 仅能看
- User Mode: 可以控制（企业版）
- Virtual Mode :每次新建一个虚拟桌面来控制
