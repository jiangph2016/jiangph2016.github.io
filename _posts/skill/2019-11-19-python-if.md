### 常规写法
```
if a>b:
    print(a)
else:
    print(b)
```

### 一行表达式
```
print(a if a>b else b)
```

### 二维列表
原理，利用二维数组的0，1作为索引
```
print([b,a][a>b])
```

### 逻辑运算
主要利用逻辑运算符的返回值进行操作，还挺绕的  

and语句

```
# 从前往后找，找假的
# 前真返后，前假返前
a and b 
```
or语句
```
# 找真的
# 前真返前，前假返后
a or b
```

所以可以这样写
```
print(a>b and a or b)
```
